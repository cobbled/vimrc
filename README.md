# vimrc

## Install Pathogen

    mkdir -p ~/.vim/{autoload,bundle}
    wget https://tpo.pe/pathogen.vim -O ~/.vim/autoload/pathogen.vim

## Insall Drawit

Download DrawIt at: [Download Link](http://www.vim.org/scripts/script.php?script_id=40)

### Install Details

    vim DrawIt.vba.gz
    :so %
    :q

### Start and stop DrawIt

    \di to start DrawIt.
    \ds to stop  DrawIt.
