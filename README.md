# VIMRC

## Install Pathogen

    mkdir -p ~/.vim/{autoload,bundle}

## Insall Drawit

Download DrawIt at: [Download Link](http://www.vim.org/scripts/script.php?script_id=40)

### Install Details

    vim DrawIt.vba.gz
    :so %
    :q

### Start and stop DrawIt

    \di to start DrawIt.
    \ds to stop  DrawIt.
